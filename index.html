Help

Shortcuts:
  • New line: ENTER
  • Strong: CTRL + B
  • Title: ALT + T
Buttons:
  • Auto-update: save all of you changes in localStorage
  • Import: import one JSON to localStorage saves
  • Export: paste current save on your console
  • New: update window with a new save (caution: if "Auto-update" is off, your current progress is lost)
  • Reset: clear all saves and restart window
